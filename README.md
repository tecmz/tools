# 魔众工具箱系统

魔众工具箱系统打造和收集各种简单、易用的在线工具大全,做出最简单方便、最干净整洁、最全面完整的在线工具,免下载的在线工具箱。

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
